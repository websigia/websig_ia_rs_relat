Objetivos
==================================

O Projeto desenvolvido tem como principal objetivo a implementação
de um WebSIG com recurso a uma solução de software e arquitetura que
apresente cartograficamente os dados durante os meses de setembro,
outubro, novembro e dezembro de 2020 relativos ao número de infetados
associados à COVID-19, por distrito. A solução desenvolvida teve por
base as linguagens HTML, CSS e JavaScript, tendo sido utilizada a
biblioteca de mapas Leaflet.
